# Hermes Frontend

This is a **plain HTML/CSS/JS + Python proxy** project. 
- No framework. No Next.js. No React. No build step.
- Entry point: `index.html` + `app.js` + `styles.css`
- Backend: `proxy.py` (Python, handles routing to 3 vLLM/Hermes agent endpoints)
- Do NOT suggest framework-based solutions.
